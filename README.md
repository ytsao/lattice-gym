# lattice-gym

Welcome to lattice-gym!

In this branch, we provide the parsers for neural network verification final project in lattice theory for parallel programming course at university of luxembourg.
Also, in lattice-exercise folder, you can check all the theoretical exercises that we have done in the exercise session.

If you have any questions, you can send the questions on Discussions or send me an email.
If you have problems with building the code, please open an issue to discuss about the problem.
